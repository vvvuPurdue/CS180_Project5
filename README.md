# CS180 Project5
## Team 15-3

Account Class:
  This class is called to hold information regarding any info in a user's account.
  Constructor:
    The constructor sets parameters equal to private variables in the class and also initializes 3 Account arrayLists. Parameters include String username, String password, String
    email, String phoneNumber, String bio, and String interests. The arrayLists initialized are friends, friendRequests, and requestedFriends. 
  getUserName:
    Gets the username.
  getPassword:
    Gets the password.
  setUserName:
    Sets the username.
  setPassword:
    Sets the password
